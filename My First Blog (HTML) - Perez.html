Historical Perspective on Computing

Looking back at time, computing goes way back to the old times. For example, the Abacus is a simple Babylonian invention that uses beads to calculate. Moving to the 19th century, Joseph Marie Jacquard made the Jacquard Machine, which inspired the following development of computers. It is an automated loom that uses punch cards to weave silk cloth automatically. Both of these inventions inspired the idea of what computing is today, the Abacus proposed the idea of creating a device to help solve computational problems while the Jacquard machine showed the potential of automating devices through programming which would influence programming and computer as today (Binary Logic).



the 1820s, Charles Babbage started his first invention called the Difference Engine which was powered by steam, this project however failed because of the lack of funding from the government. Years later, he made his second invention called Analytical Engine which has been discussed as the first programmable computer using punch cards. At this time, Charles met Ada Lovelace (Deemed to be the first programmer), Lovelace became interested in the project of Charles Babbage which she helped by giving her vision of how the Analytical Engine can be programmed to solve Bernoullis Number and the potential of what programming could bring to the future.



In 1942, John Atanasoff and his student Clifford Berry made the first digital computer which replaced mechanical parts with electronic parts, primarily, vacuum tubes which made computations much faster. The ABC Machine used a Base-2 binary system instead of the traditional, base-10 or decimal system that would become the foundation for todays programming.



1939-1945, 1942, The Colossus Mark I is considered to be one of the earliest slightly programable electronic digital computers were invented in collaboration with mathematician and logician, Alan Turing, (Considered to be the father of Modern Computer Science and AI, who also rensponsible for deciphering the enigma code using his invention called Bombe) deciphered the German encrypted messages during World War II. Alan's invention, the Turing Machine which can stimulate algorithms, and his contribution to cryptography, greatly contributed to the development of modern computers and programming.



In 1945, the ENIAC (Electronic Number Integrator and Computer) was invented by John Mauchly and J. Presert Ecker. The ENIAC was the first general-purpose electronic digital computer that at that time showed computational prowess which would inspire machines for the following years. Around this time, Jon Von Neumann an American Mathematician, proposed the Jon Neumann Architecture which founded how computers of today were built. In addition, he helped in creating the EDVAC (Electronic Discrete Variable Automatic Computer) which has a stored program or instructions in the computer's memory, combining this with Neumann's Von Neumann Architecture, became the main principle of designing modern computers.



In 1947, Transistors were invented which replaced vacuum tubes as it is more efficient, consumed less power, and also smaller. This also led the way for High-Level programming languages such as Fortran, started to be used because of the transistor's much better computation power and efficiency especially when they created an integrated circuit where transistors were packed into one single circuit. Because of the increase in computation power, and high-level programming language, Fortran, started to become widely used by people to program applications to aid in solving computational problems.



During the 1960s - 1990's, the ARPANET paved the way for what we use today, theInternet, which connects different computers and can transmit information from one computer to another. Cloud computing is another example where a person can store information on the internet.

Introduction to IT Basics

Hardware

- Hardware is a physical component of a computer that is used to either enter data using an input device(e.g.Keyboard, Mouse, Microphone,) or present the product of the data by an output device (e.g. Speaker, Monitor, Printer). The input data that was entered by the input device will then be stored and processed by the processing components of the computer (e.g. RAM, CPU, GPU, ssd/hard disk,) which would then present or display the output data/product using an output device.

Software

- Software is an application that is a set of programmed code that acts as instructions for the computer to follow. There are two types of software applications 1. Application Software and 2. Systems Software. [APPLICATION SOFTWARE] is used to make everyday life faster and more efficient compared to the traditional ways of doing tasks which would take a longer time compared to using an application. [SYSTEMS SOFTWARE] focuses on the operating system of the computer (e.g. windows, macOS, Linux), drivers, and utilities. The primary purpose of the operating system is to help manage computer data and help software communicate with hardware.

Networking

- Networking is the connection between computers to send, share, or store information/resources from one computer(e.g. personal desktop) to another(e.g. mobile phone, internet, server,) which could be done physically using a cable or wirelessly by signals.

Role of IT in modern organizations

- IT generally reduces our workload by making tasks automated which makes accomplishing faster and more efficient, but what is its role in modern organizations of today? Today organizations use Information Technology to increase productivity and efficiency in work enabling the organization to allocate some of the resources to other work. Data management along with Cybersecurity is another example that the biggest organization has. With the large volume of data that the big organizations gather, store, and process, they would have to protect it from getting stolen from hacking or data breaching by enforcing a cyber security system as it is a valuable asset of the organization, it would take them a long time if they were to manually compute all of the data so they implement the uses of Information technology in their work to ensure progress.
